# Student Dashboard Design System

## Brand Identity

Modern productivity application inspired by Notion, Linear, and Todoist.

Design Goals:

- Clean
- Minimal
- Professional
- Student-focused
- Responsive
- Accessible

---

## Color Palette

### Primary (Indigo)

Used for:

- Primary buttons
- Active navigation items
- Links
- Focus rings

```txt
primary-50
primary-100
primary-200
primary-300
primary-400
primary-500
primary-600
primary-700
```

### Neutral (Slate)

Used for:

- Backgrounds
- Cards
- Borders
- Text

```txt
neutral-50
neutral-100
neutral-200
neutral-300
neutral-400
neutral-500
neutral-600
neutral-700
```

### Success (Emerald)

Used for:

- Completed tasks
- Success notifications

### Warning (Amber)

Used for:

- Pending tasks
- Due soon

### Danger (Red)

Used for:

- Delete actions
- Errors
